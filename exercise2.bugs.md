 **Предшествующие условия:**

 - Проверить корректность кода состояния HTTP
 - Проверить тело и структуру ответа 

 **Ожидаемый результат:** 
 - HTTP статус: 201 Created
 - Тело ответа в JSON-формате с созданными данными возвращается от сервера
